# Contributing

Thank you for considering contributing to this Swift package! We welcome bug reports, feature suggestions, and code improvements.

## Getting Started

1. Fork the repository and clone your fork locally.
2. Create a feature branch for your work (`git checkout -b feature/your-feature`).

## Code Formatting

This project uses Apple's `swift-format` tool. **All code must follow the rules defined in the `.swift-format` file** at the root of the repository.

Before committing, always format your changes:

```bash
swift format --in-place .
```

We recommend configuring your editor (Xcode, VS Code, etc.) to format automatically on save.

## Running Tests

Make sure everything still works:

```bash
swift test
```

## Submitting Changes

1. Ensure your code is formatted and all tests pass.
2. Write clear commit messages.
3. Open a pull request with a short description of your changes.

We review every contribution quickly and appreciate your help making the package better!
